Project Descriptions

For the past 5 years I have developed many projects, all of which feature something that I never tried before. I can make applications that range from games to one that you use in the workplace. Below are projects that I have done and a brief description.

VR-Physics


VR-Physics is a virtual reality physics game that was inspired by previous work brought on by Benjamin Brown's Thesis. This took some elements taken from his thesis and allowed for an educational game that was fun and engaging for the user.

VR-Physics Website


This is a website that showcases VR-Physics for any visitor interested in the product. This website was built by myself and uses some functionality provided by HTML5UP.

Visit the website by following this link here -> Proceed to VR-Physics!

Angry Birds Replica


This is a game that had a soul purpose of being able to manipulate and use a library (in this case a physics library) and use it to my advantage. This included gravity, physics hit detections, and using background information to determine when certain events happen.

Battleboxes - Enhanced Deluxe Edition


This is a game that allows two players to competitively play against eachother in a contained environment. I built this game to be able to run two separate instances simultaneously and have no input delays.

Mario Pinball (All Art Owned By Nintendo)


This is a graphics project in which I was included in a team that had various aspects like: lighting, physics, and it is a playable game. Parts of this project were challenging. One of these parts was the Bullet Physics Library. This was challenging because I had to understand code that I did not read and it was horribly documented. This is important in software engineering because you need to know how to read and understand other's code.

Void Realm


Void Realm is a game that features mechanics inspired from multiple games. This is a game that uses the Unity 2017 Engine and features a top-down perspective. For those who dont know Unity, this engine is not really great for top down 2D games. This project is an attempt at using engine for a top down game and take mechanics from Fallout and Legend of Zelda. These mechanics include an in-game terminal, Zelda style dungeons, and powers to use during your journey.

Shape Bros


This is a project where the premise was to allow for multiple playable characters. This game is inspired by games like A Way Out, Brothers: A Tale of Two Sons, and Thomas Was Alone. The game is also meant to be an educational experience where shapes take a huge place in the puzzles.

My No Name Game (Mobile Only)


My No Name Game is exactly what it sounds like. A game that doesn't have a name! I used assets created by myself from Battleboxes: Deluxe Enhanced Edition. This game is my first android application and it worked for multiple devices like: Samsung Galexy S4/S5/S6/S7, Note 1/2 (XL included), and Nexus 5.

Team 7 Perspective Puzzles


This was the second virtual reality game that I was part of. This game features puzzles that are purely based on perspective. One puzzle is where you can make yourself the size of a mouse or the size of a giant to navigate a house searching for keys. The next puzzle is when you use a flashlight to cast shadows that make a code.
